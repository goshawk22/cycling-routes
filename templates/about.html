About

The aim of this website is to provide a platform for members of WUCT to share and discover cycling routes.

Technical Information

This site is built using Flask, a Python web framework. It uses a SQLite database to store route data. It's currently hosted on a STARDUST1-S VPS from Scaleway. It only has an IPV6 address so it uses a cloudflare tunnel to expose the site to the internet. Backups are made every 8 hours using a systemd timer and service. The script zips the database file and GPX files and uploads them to a Cloudflare R2 bucket.

Source Code

The source code for this site is available on GitHub: https://github.com/goshawk22/cycling-routes

Feel free to contribute or report issues. Contributions are welcome!
